auto bolson::parse::opae::TripParser::Parse ( const std::vector< illex::JSONBuffer * > &  buffers_in,
std::vector< ParsedBatch > *  batches_out 
) -> Status
overridevirtual

Parse buffers containing raw JSON data.

Appends parsed buffers as RecordBatches to batches_out. No guarantees are made about the relation between the input buffers and output batches, other than that for each valid JSON object, there will be one corresponding Arrow record in one of the resulting batches.

Parameters
buffers_inThe buffers with the raw JSON data.
batches_outThe parsed data represented as Arrow RecordBatches.
Returns
Status::OK() if successful, some error otherwise.
